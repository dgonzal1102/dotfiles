## Configure Linux

- From home directory run: git clone https://github.com/dgonzal1102/dotfiles.git
- cd dotfiles/
- sudo rm ~/.zshrc
- stow zshrc
- stow backgrounds
- stow fastfetch
- stow kitty
- stow rofi
- stow scripts
- stow matugen
- stow waybar
- sudo rm ~/.config/hypr/hyprland.conf
- stow hypr
- hyprctl reload
- cd dotfiles/sddm/
- sudo cp -r sddm /usr/share
- sudo cp -r sddm.conf.d /etc
- sudo reboot
